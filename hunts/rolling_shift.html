Rolling Shift

Shift each character in this string back by it's index in the string (not counting spaces):

"mqro ztkma dsq bfpdxw vbveo gm oog rj ynl arnp mespi zf mbz hllf eoilrjkyqwq caba"


Example

The string "sgg djhyi" would become "red zebra"

s -> (shift back 1) -> r
g -> (shift back 2) -> e
g -> (shift back 3) -> d

d -> (shift back 4) -> z (shift around the end of the alphabet to z)
j -> (shift back 5) -> e
h -> (shift back 6) -> b
y -> (shift back 7) -> r
i -> (shift back 8) -> a
